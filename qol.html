Quality of Life Improvements and Tweaks

Creating a Separator in MO2
  1. Right-click anywhere in the left pane of MO2 and select Create Separator
  2. Name the separator Tweaks

Quests Are In Skyrim

Installation instructions:

  • Main File - QuestsAreInSkyrim_USSEP_1_2_reupload 1.2

- Fixes base-game radiant quests being generated in DLC areas

Dragonborn Delayed

Installation instructions:

  • Main File - 1 - Dragonborn Delayed - Horn of Jurgen Windcaller 1.0

- Delays the start of the Dragonborn DLC until after you finish the Horn of Jurgen Windcaller quest

Dawnguard Delayed

Installation instructions:

  • Main File - Dawnguard Delayed - Level 30 1.0

- Delays the start of the Dawnguard DLC until reaching level 30

Realistic Conversations

Installation instructions:

  • Optional File - Realistic Conversations - longer pause between sentences 2.1

- Prevents NPCs from constantly barraging you with greetings

Quest NPCs Run

Installation instructions:

  • Main File - Quest NPCs Run 1.12

- Makes slow walking quest NPCs you follow run

Bounty Preview

Installation instructions:

  • Main File - Bounty Preview 2.0.0

- Allows you to preview bounty missions from innkeepers and either accept/reject them

All Thieves Guild Jobs Concurrently

Installation instructions:

  • Main File - All Thieves Guild Jobs Concurrently - USSEP Version 1.3

- Allows you to accept all seven types of Thieves' Guild missions at once, greatly reducing the amount of time it takes to complete them all

Simple Offence Suppression

Installation instructions:

  • Main File - Simple Offence Suppression 2.0

- Prevents friendly NPCs from turning hostile on you if you accidentally hit them during combat

Encounter Zones Unlocked

Installation instructions:

  • Old File - Encounter Zones Unlocked SE 1.4
IMPORTANT: Make sure you download the Old File rather than the Main File, as that is the only version supported by the version of the game used in the guide.

- Prevents the game from locking an area's encounter zone level when you first enter it
