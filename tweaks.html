Tweaks

Creating a Separator in MO2.

  1. Right-click the empty space in the left pane of MO2 and select Create Separator.
  2. Name the separator Tweaks.

Survival Control Panel

Installation instructions:

  • Main Files - Survival Control Panel
Allows customizing native Survival Mode features like Sleep to Level Up.

Quests Are In Skyrim

Installation instructions:

  • Main Files - QuestsAreInSkyrim_USSEP_1_3
  • Quests Are In Skyrim - USSEP Patch
Fixes base-game radiant quests being generated in DLC areas.

Timing is Everything

Installation instructions:

  • Main Files - Timing is Everything SE
  • Timing is Everything SE - Settings Loader
  • Timing is Everything DBF Configuration
Allows you to customize the starting conditions for various quests.

Quest NPCs Run

Installation instructions:

  • Main Files - Quest NPCs Run
Makes slow walking quest NPCs you follow run.

Bounty Preview

Installation instructions:

  • Main Files - Bounty Preview
Allows you to preview bounty missions from innkeepers and either accept or reject them.

Simple Offence Suppression

Installation instructions:

  1. Main Files - Simple Offence Suppression SE
  2. You will be prompted with a FOMOD installer, click Install
Prevents friendly NPCs from turning hostile on you if you accidentally hit them during combat.

Encounter Zones Unlocked

Installation instructions:

  • Main Files - Encounter Zones Unlocked AE
Prevents the game from locking an area's encounter zone level when you first enter it.

To Your Face SE - AE - VR

Installation instructions:

  • Main Files - To Your Face AE
Prevents NPCs from making random greetings unless they are in front of the player.

Enhanced Invisibility

Installation instructions:

  1. Main Files - Enhanced Invisibility
  2. You will be prompted with a FOMOD installer, click Install
SKSE plugin that improves and fixes bugs with Invisibility effects.

Fires Hurt RE

Installation instructions:

  • Main Files - FiresHurtRE
Now fires, campfires, embers, steam & magic pillars will hurt.

Fire Hurts NG

Installation instructions:

  • Main Files - Fire Hurts NG
Optimizes the previous mod and makes it compatible with all game versions.

Animals Swim (Sort of)

Installation instructions:

  • Main Files - Animals Swim (Sort Of)
Rabbits (and other animals) won't walk at the bottom of the sea anymore, they will float near the surface instead.

Dynamic Collision Adjustment

Installation instructions:

  • Main Files - Dynamic Collision Adjustment
Adjusts the character controller collision while sneaking and swimming so you can fit under objects.

Remember Lockpick Angle - Updated

Installation instructions:

  • Main Files - Remember Lockpick Angle
After breaking a lock-pick, the new lock-pick is placed at the same angle where the last one broke.

Better Jumping SE

Installation instructions:

  • Main Files - Better Jumping AE
Allows to jump while sprinting.

Instantly Skip Dialogue NG

Installation instructions:

  • Main Files - Instantly Skip Dialogue NG
Allows the player to skip dialogue instantly, bypassing the 750 millisecond delay. This includes dialogue that's normally unskippable.

Multiple Floors Sandboxing

Installation instructions:

  • Main Files - Multiple Floors Sandboxing
Allows NPCs to climb stairs around them and explore their surroundings more.
