Tweaks

Creating a Separator in MO2

  1. Right-click the empty space in the left pane of MO2 and select Create separator.
  2. Name the separator Tweaks.

Better Jumping SE

Allows to jump while sprinting.

Installation instructions

  • Main Files - Better Jumping (1.6.X)

Better Third Person Selection - BTPS

Makes it much easier to select objects in third person, including when on horseback.

Installation instructions

  • JContainers SE
    • Main Files - JContainers SE OR JContainers GOG, based on your game version
  • Main Files - Better Third Person Selection (AE - SE)

Tavern AI fix

Waiters will only serve you if they can see you, and innkeepers will show you to your rented room once initially, but you can ask them again for directions if needed.

Installation instructions

  • Main Files - TavernAIFix

Remember Lockpick Angle - Updated

After breaking a lockpick, the angle will be remembered for the next attempt.

Installation instructions

  • Main Files - Remember Lockpick Angle

To Your Face SE - AE - VR

Prevents NPCs from making random greetings unless they are in front of the player.

Installation instructions

  • Main Files - To Your Face AE

Simple Offence Suppression

Prevents neutral NPCs from turning hostile when you accidentally hit them.

Installation instructions

  • Main Files - Simple Offence Suppression
    • You will be prompted with a FOMOD installer:
      1. Install

Sensible Oculory Solution - Logical Mzark Puzzle - Discerning The Transmundane - Main Quest

Makes the Mzard puzzle less confusing by giving you visual hints on how to solve it.

Installation instructions

  • Main Files - Sensible Oculory Solution - Logical Mzark Puzzle

Simple Dual Sheath

Makes equipped left hand weapons and both staves visible on the player or NPCs. Also shows shields on the back.

Installation instructions

  • Main Files - Simple Dual Sheath for 1.6.629 and newer
    1. Once the mod has been installed, double-click it in the left pane of MO2.
    2. In the INI Files tab, select SKSE/Plugins/SimpleDualSheath.ini
    3. Set Flags (line 26) to FirstPerson|Player|NPC.
