Visuals

Creating a Separator in MO2.

  1. Right-click the empty space in the left pane of MO2 and select Create Separator.
  2. Name the separator Visuals.

Static Mesh Improvement Mod

Installation instructions:

  1. Main Files - SMIM SE 2-08
  2. You will be prompted with a FOMOD installer, click Install
A massive project to greatly improve the appearance of countless static 3D models.

Blended Roads

Installation instructions:

  1. Main Files - Blended Roads
  2. You will be prompted with a FOMOD installer, select Really Blended Roads in both sections and then Install
Better blends roads into the landscape textures to avoid seams and gives them 3D meshes to look more natural.

Assorted Mesh Fixes

Installation instructions:

  • Main Files - Assorted mesh fixes
Fixes various issues with vanilla Skyrim meshes.

Mesh Patch for Various Mods

Installation instructions:

  • Main Files - Assorted Mesh Fixes - SMIM - Really Blended Roads Patch
Patch between Assorted Mesh Fixes, SMIM and Really Blended Roads.

Water Effects Brightness and Reflection Fix

Installation instructions:

  1. Main Files - Water Effects Brightness and Reflection Fix
  2. You will be prompted with a FOMOD installer, click Install
Fixes bright water effects at night and neon green reflections.

Simplicity of Sea

Installation instructions:

  1. Main Files - Water Mod
  2. You will be prompted with a FOMOD installer, select 2K Textures and then Install
A light-weight water mod. Includes ENB-compatible displacement maps. An optional ESP only edits water records.

Noble Skyrim

Installation instructions:

  • Main Files - A. Noble Skyrim - FULL PACK_2K
  • Update Files - NSM - SMIM Patch
Retextures most of the game without going overboard. This is a good base if you want to add your own retextures to the game.

Security Overhaul SKSE - Lock Variations

Installation instructions:

  1. Main Files - Security Overhaul SKSE - Lock Variations
  2. You will be prompted with a FOMOD installer, select 1K Textures and then Install
SKSE based framework for replacing overused lock model with unique lock variants.

Security Overhaul SKSE - Regional Locks

Installation instructions:

  1. Main Files - Security Overhaul SKSE - Regional Locks
  2. You will be prompted with a FOMOD installer, select 1K Textures and then Install
Regional variant lock models for Security Overhaul SKSE - Lock Variations.

Security Overhaul SKSE - Lock Add-ons

Installation instructions:

  1. Main Files - Security Overhaul SKSE - Add-ons
  2. You will be prompted with a FOMOD installer, select 1K Textures and then Install
Additional variant lock models for Security Overhaul SKSE - Lock Variations.

Soft Shadows

Installation instructions:

  • Main Files - Soft Shadows
SKSE plugin which dynamically adjusts the look of shadows depending on lighting conditions.

Twilight

Installation instructions:

  • Main Files - Twilight
SKSE plugin which simulates twilight (dawn and dusk) for any weather.

Community Shaders

Installation instructions:

  • d3dcompiler_47
  • From the downloaded archive, extract everything to the game's Root folder.
  • Main Files - Community Shaders
  • Tree LOD Lighting
  • Grass Lighting
  • Grass Collision
  • Screen-Space Shadows
  • Water Blending
Fixes bugs and improves visuals introducing new and optimized rendering techniques.
The first game session with CS will have to cache shaders so the initial loading will take considerably longer, do not be alarmed by this and keep in mind it is a 1 time thing.

Grass Sampler Fix

Installation instructions:

  • Main Files - Grass Sampler Fix
Improves the look of grass by forcing it to use anisotropic filtering.
