Community Shaders

Community Shaders is a framework that fixes rendering bugs and allows adding new graphics features through addons. The performance impact depends on which modules are used in addition to the main one, which, by itself, improves performance.

CS requires a DX11.1 compatible GPU, you can check if yours is compatible by looking it up here, then refer to the Graphics Features list.

Installation instructions

  • Main Files - Community Shaders
  • A Dragonborn's Fate Resources
    • Main Files - Community Shaders - DBF Configuration

The DBF Configuration file applies the following settings:

  • FSR 3.1 AA as anti-aliasing method, replacing TAA and massively improving visual clarity for a tiny performance hit.
    • If you are using an RTX GPU, you should switch from FSR 3.1 AA to DLAA, which gives a similar result with better performance.
  • Increased sample count for Screen Space Shadows.
  • SSGI set to quarter resolution - the full and half resolution options are too heavy to be recommended on most systems.

Recommended Addons:

  • Grass Lighting - Improves grass visuals with directional lighting, subsurface scattering, and fixes UV-mapping issues for complex grass. Includes normal and specular map support.
  • Grass Collision - Makes grass collide with the player and NPCs.
  • Screen Space Shadows - Adds extra shadow details to objects, plants, and your character that wouldn't normally cast shadows
  • Light Limit Fix - Removes the game's light limit (not the shadow limit) while adding particle lights and contact shadows.
  • Subsurface Scattering - Adds screen-space subsurface scattering to characters to simulate realistic skin.
  • Wetness Effects - Rainfall creates realistic wet surfaces, puddles that persist and evaporate over time, and dynamic ripple effects, while precipitation occlusion ensures raindrop visuals appear only on exposed areas.
  • Water Effects - Simulates water caustic effects and adds optimised parallax support.
  • Terrain Shadows - Enables rough terrain shadows with infinite distance, in game and on world maps.

    • Unless you are generating new heightmaps (part of the LOD Guide, linked right after this section), install the optional file as well!

  • Cloud Shadows - Allows clouds to cast shadows.
  • Skylighting - Simulates large scale world-space ambient occlusion.

Each feature, save a few, has a different performance impact - the heaviest ones are Screen Space Shadows, SSGI, AA replacement, and Light Limit Fix. The rest are close to negligible.
