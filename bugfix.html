Creating a Separator in MO2.

  1. Right-click the empty space in the left pane of MO2 and select Create Separator.
  2. Name the separator Bug Fixes.

Engine Fixes

Installation instructions:

  1. Download and install the Main Files - (Part 1) SSE Engine Fixes for 1.6.629 and newer
  2. Download the Main Files - (Part 2) Engine Fixes - skse64 Preloader and TBB Lib 2020.3 file using the Manual Download button
  3. From the downloaded archive, extract everything to the game's Root folder
  4. Once the files are installed, right-click on the mod in the left pane of MO2 and select Open in Explorer
  5. Navigate to SKSE\Plugins and open EngineFixes.toml in a text editor (such as notepad)
    1. Set CleanSKSECosaves to true (line 5)
    2. Set MaxStdio to 8192 (line 11)
    3. Set SleepWaitTime to true (line 18)
    4. Set SaveGameMaxSize to true (line 66)
Fixes numerous bugs within the game's engine.
