SSE Engine Fixes (skse64 plugin)

A collection of engine bug fixes and tweaks.

Installation instructions

  • Main Files - (Part 1) SSE Engine Fixes for 1.6.1170 and newer
    1. Once the mod has been installed, double-click it in the left pane of MO2.
    2. In the Text Files tab, select SKSE/Plugins/EngineFixes.toml
    3. Set CleanSKSECosaves (line 5) to true.
    4. Set MaxStdio (line 11) to 8192.
  • Main Files - (Part 2) Engine Fixes - skse64 Preloader and TBB Lib
    1. Open the downloaded archive and move everything to the game's Root folder.
