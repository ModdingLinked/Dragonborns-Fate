Gameplay

Creating a Separator in MO2.

  1. Right-click the empty space in the left pane of MO2 and select Create separator.
  2. Name the separator Gameplay.
It is especially important to read the descriptions of these mods. If you do want to use the SimonRim suite, it is recommended to install the whole package to avoid missing dependencies and to have the most comprehensive overhaul.

Survival Mode Improved - SKSE

Installation instructions:

  • Main Files - Survival Mode Improved - SKSE
SKSE plugin implementation of Skyrim's Survival Mode.

Mysticism - A Magic Overhaul

Installation instructions:

  • Main Files - Mysticism - A Magic Overhaul
  • Optional Files - Mysticism - Jump Spell Addon
Complete overhaul of the magic system, designed to balance existing spells and add powerful new magical effects.

Aetherius - A Race Overhaul

Installation instructions:

  • Main Files - Aetherius - A Race Overhaul
  • You will be prompted with a FOMOD installer. Select following:
    1. Install
Complete overhaul of human, elf and beast races designed to balance the races and add powerful new gameplay options.

Adamant - A Perk Overhaul

Installation instructions:

  • Main Files - Adamant - A Perk Overhaul
  • Optional Files - Adamant - Bard Perks Addon
  • Optional Files - Simonrim Attack Speed Fix
  • Improvement Names Customized AE (make sure this loads above the Smithing Addon, which has to win the file conflict)
  • Optional Files - Adamant - Smithing Addon
Complete overhaul of the perk trees designed to balance existing skills and add powerful new perks.

Mundus - A Standing Stone Overhaul

Installation instructions:

Thaumaturgy - An Enchanting Overhaul

  • Main Files - Thaumaturgy - An Enchanting Overhaul

Hand to Hand - An Adamant Addon

Installation instructions:

  • Main Files - Hand to Hand - An Adamant Addon
  • Optional Files - Hand to Hand - Locksmith Addon
This addon will turn the functionality of the Remember Lockpick Angle mod into a perk. You can skip this if you wish to always remember the lockpick's position.
  • Optional Files - Hand to Hand - Trainers and Skill Books Addon
Addon for Adamant - A Perk Overhaul that merges Lockpicking and Pickpocketing into a single skill and adds a new perk tree for Hand to Hand combat.

Stormcrown - A Shout Overhaul

  • Main Files - Stormcrown - A Shout Overhaul
Overhauls and rebalances the Shout system, adds new Shouts and provides additional progression for the player's experience as Dragonborn.

Journeyman - A Fast Travel Overhaul

Installation instructions:

  • Main Files - Journeyman - A Fast Travel Overhaul
If you are ok with needing fast travel kits to use fast travel even outside of Survival mode, then you can skip the next steps and jump to the next mod.
  1. Once the files are installed, right-click on the mod in the left pane of MO2 and select Open in Explorer
  2. Navigate to SKSE\Plugins and open Journeyman.ini in a text editor (such as notepad)
  3. Set bEnableOnlyOnSurvivalEnabled to true (line 3)
Journeyman is a lightweight, highly compatible fast travel overhaul that restricts the player from fast traveling unless they have prepared a travel pack.

Sorcerer - A Staff and Scroll Overhaul

Installation instructions:

  • Main Files - Sorcerer - A Staff and Scroll Overhaul

Gourmet - A Cooking Overhaul

Starfrost - A Survival Overhaul

Candlehearth - An Inn Overhaul

  • Main Files - Artificer - An Artifact Overhaul

Sensible Bribes

Installation instructions:
