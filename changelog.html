Changelog (Month Day)

2025

May 12

Whole Guide

  • Added support for GOG.

Bug Fixes

  • Added the following mods:
    • DEEJMASTER's Mini Mods Tweaks and Fixes
    • Skyrim Misc Bugfix Compilation
    • Dragon Leveled List Fixes

User Interface

  • Added QuickLoot IE - A QuickLoot EE Fork and its requirement, PapyrusUtil SE. Also added the QuickLoot IE - Inventory style retexture for it.

Tweaks

  • Added Better Third Person Selection - BTPS and its requirement, JContainers.

Visuals

  • Added No Camera Attached Smoke or Fog for users of CS Particle Patch, improving performance.
  • Added a patch between Simple Lighting Overhaul and the newly added Skyrim Misc Bugfix Compilation.

Extended Finish

  • Documented the changes applied by the DBF CS configuration.
  • Added a note about switching to DLAA for extra performance if you have an RTX GPU.

LOD Guide

  • Updated the Whiterun comparison images and added one to showcase a child worldspace (from Dragonsreach).
  • Regenerated the DBF Pre-Generated Terrain LOD, it used LOD32 512 resolutions instead of 256. You don't need to do reinstall the file, as it changed almost nothing in practice (reason why we just use the 256 default).
  • Switched from DynDOLOD DLL SE to NG, and added steps to clean the base game plugins so that the large refs workaround bug in DynDOLOD can be used, as it was still present with the previous setup.
  • Updated DynDOLOD settings for greater detail.
  • DBF Pre-Generated Object LOD was updated to reflect the 2 changes above.

May 10

Utilities

LOD Guide

  • Added a missing step to disable the Temp Snow Replacement after terrain LOD generation, don't forget this!

ESL Guide

  • Mentioned the 2 known bugs with ESL cells.
